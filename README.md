# Ai00-X - AI驱动桌宠游戏

## 项目简介

Ai00-X 是一款创新的AI驱动桌宠游戏，旨在为用户提供真正智能的数字伙伴体验。每个Ai00-X都是独一无二的AI生命体，能够学习、成长，并与用户建立深度的情感连接。
